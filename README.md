## 🎯 Overview

Evidence-based safety reference for tattoo, piercing, and PMU during pregnancy, breastfeeding, and TTC. All trimesters with specific risk rationale.

**Category:** Client Health & Aftercare

This is a free, open-source widget from the [Poli International Widget Suite](https://poliinternational.com/tools/) - a collection of practical tools for tattoo and piercing studios, artists, and clients. It runs entirely in the browser; no data is sent to a server.

---

## ✨ Features

- ✅ **Tattoo During Pregnancy**
- ✅ **Piercing While Pregnant**
- ✅ **PMU Pregnancy Safety**
- ✅ **Breastfeeding Tattoo**
- ✅ **Body Art Pregnancy Risk**
- ✅ **Can I Get Pierced While Pregnant**
- ✅ **Responsive Design** - works on desktop, tablet, and mobile.
- ✅ **Free & Open Source** - MIT licensed.

## 🚀 Installation

### Option 1: Use Online (Recommended)

No installation needed! Just visit:
**[https://poliinternational.com/tools/pregnancy-piercing-safety/](https://poliinternational.com/tools/pregnancy-piercing-safety/)**

### Option 2: Run Locally

This tool is pure HTML/CSS/JS - no build step. Clone the repo and open `index.html`:

```bash
git clone https://github.com/Poli-International/pregnancy-piercing-safety.git
cd pregnancy-piercing-safety
# open index.html in your browser
```
